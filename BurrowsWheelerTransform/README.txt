To Run Program:
	The program's main file is main.py
	example:
	from the command line: python main.py bwt sonnet1.fasta sonnet1.bwt.fasta

The program should print what it is reading, the length of the string, and where to check for the results.

The output file adds 'BWT: ' or 'iBWT: ' plus the original first line of the file
